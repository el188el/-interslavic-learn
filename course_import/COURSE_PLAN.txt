================================================================================
ПЛАН КУРСА Medžuslovjansky (консультант + дорожная карта контента)
================================================================================

Цели (зафиксировано с автором):
  • Языки объяснений: русский и английский.
  • Ярлык межславянского в интерфейсе: по умолчанию латиница (title_isv_lat).
  • Задача ученика: свободнее общаться со славянами в поездках, меньше страха «не
    понимать и быть непонятым»; увереннее входить в сообщество межславянского языка.

--------------------------------------------------------------------------------
ФИЛОСОФИЯ ПОСТРОЕНИЯ (плотный курс, дух Duolingo без воды)
--------------------------------------------------------------------------------
  1. Каждый урок: короткая теория + таблица лексики / грамматики + нюанс (tip) +
     минимум 4 упражнения: соответствие, выбор, пропуск, ввод слова.
  2. Лексика — ситуационная (граница, транспорт, еда, здоровье, работа, досуг).
  3. Грамматика — по ступеням: настоящее → падежи блоками → прошлое/будущее →
     сложное предложение → стиль и уместность.
  4. Блок «мосты»: сознательное сопоставление с польским/чешским/болгарским и т.д.
     не как отдельный «академический» курс, а как «что может означать корень».

--------------------------------------------------------------------------------
СТРУКТУРА УРОВНЕЙ (CEFR-ориентир)
--------------------------------------------------------------------------------
  A0–A1  Алфавит, чтение, приветствия, числа, время, цвета, семья, дом, еда.
  A2     Путешествие, покупки, здоровье, экстренные фразы, город, работа, досуг.
  B1     Глаголы движения и вид, падежи (род. / дат. / твор. / предл.), сложный синтаксис.
  B2     Порядок слов, стиль, споры и мнения вежливо, «на границе», онлайн-сообщество.
  C1–C2  Нюансы близкие к носителю: идиомы, регистры, устойчивые образы (по мере готовности).

--------------------------------------------------------------------------------
ЧТО УЖЕ ЕСТЬ В ГЕНЕРАТОРЕ (tool/course_gen/, см. full_course.json после сборки)
--------------------------------------------------------------------------------
  Реализованы уроки в части категорий (введение, алфавит, приветствия, числа в три
  урока, время, цвета, семья, еда, глаголы наст., падежи ном/вин., прошлое,
  будущее, предлоги, стиль, блок «рядом с носителем»). Остальные категории из
  syllabus.py пока БЕЗ уроков — в приложении они видны, но с подписью «Скоро».

--------------------------------------------------------------------------------
ЧТО ДОБАВИТЬ ДАЛЬШЕ (приоритетный бэклог для наполнения)
--------------------------------------------------------------------------------
  Высокий приоритет (практика «за границей» и сообщество):
    • cat_fc_pronunciation — детальнее звуки č/š/ž/nj и ударение.
    • cat_fc_calendar — дни недели, месяцы, даты в разговоре.
    • cat_fc_travel, cat_fc_directions, cat_fc_accommodation — полные диалоги.
    • cat_fc_shopping, cat_fc_emergency, cat_fc_health — фразы без воды.
    • cat_fc_border_talk — граница, дорога, объяснение цели поездки.
    • cat_fc_community, cat_fc_online — чаты, правила вежливости, мероприятия.

  Средний приоритет (грамматика для устойчивой речи):
    • cat_fc_cases_gen_dat, cat_fc_cases_ins_prep — полные парадигмы и типичные ошибки.
    • cat_fc_verbs_motion, cat_fc_aspect — пары сов./несов., «идти/ехать».
    • cat_fc_imperative, cat_fc_complex_sentence — просьбы, запреты, союзы.

  Ниже приоритет (углубление):
    • cat_fc_slavic_bridges — осознанные различия PL/CZ/SR/BG рядом с ISV.
    • cat_fc_native — расширение идиоматики и регистров.

--------------------------------------------------------------------------------
КАК РАБОТАТЬ ВМЕСТЕ С АССИСТЕНТОМ
--------------------------------------------------------------------------------
  1. Расширять curriculum_extend.py новыми lesson(...) по одной категории за раз.
  2. Запускать: python tool/generate_course_import.py && python tool/export_course_import_to_sql.py
  3. Импорт SQL в Supabase; категории без уроков уже созданы — меняется только
     появление новых строк в course_lessons.

Конец файла плана (обновляйте по мере заполнения уроков).
