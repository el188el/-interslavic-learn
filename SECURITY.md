# Security

If you find a vulnerability in this repository or the published app, please report it responsibly.

- Open a **private** security advisory on GitHub (preferred), or  
- Email the maintainers using the contact address in the app (Settings → resources / about), if available.

Please do not open public issues for undisclosed security problems.

Do **not** commit API keys, Supabase service keys, signing keystores, or `dart_defines.json` with secrets. Use `dart_defines.example.json` and local untracked copies only.
